# Data Dictionary

Dataset: Premier League 2025/26 Match Data

Source:
Football-data.co.uk

## Match Information

| Column | Meaning |
|---|---|
| Div | Division |
| Date | Match date |
| HomeTeam | Home team |
| AwayTeam | Away team |
| FTHG | Full-time home goals |
| FTAG | Full-time away goals |
| FTR | Full-time result |
| HTHG | Half-time home goals |
| HTAG | Half-time away goals |
| HTR | Half-time result |

## Match Statistics

| Column | Meaning |
|---|---|
| HS | Home shots |
| AS | Away shots |
| HST | Home shots on target |
| AST | Away shots on target |
| HC | Home corners |
| AC | Away corners |

## Result Values

H = Home Win  
D = Draw  
A = Away Win
